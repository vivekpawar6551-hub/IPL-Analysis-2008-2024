# IPL Analysis Dashboard (2008–2024)

Interactive Power BI dashboard built using IPL data from 2008 to 2024. This dashboard provides insights into team performance, player statistics, tournament records, and match analysis.

## Features
- Tournament Winner Analysis
- Total Runs, Wickets, 4's & 6's Overview
- Orange Cap & Purple Cap Statistics
- Batsman Performance Analysis
- Bowler Performance Analysis
- Toss Decision Winning Percentage
- Top 5 Batsmen
- Top 5 Bowlers
- Top Winning Teams
- Top 10 Player of the Match Awards

## Tools Used
- Power BI
- DAX
- Power Query

## Insights
- Virat Kohli scored the highest runs in IPL history
- YS Chahal emerged as one of the leading wicket takers
- Mumbai Indians and Chennai Super Kings dominated tournament wins
- Field-first toss decisions resulted in higher winning percentages
- Performance trends analyzed across IPL seasons from 2008–2024
